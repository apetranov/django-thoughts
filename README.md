# How to run locally:

## Step 1: clone repo
```bash
git clone https://github.com/apetranov/django-thoughts.git
```
## Step 2: create virtual environment
```bash
python -m venv venv
```
## Step 3: activate virtual environment
### Windows
```bash
venv\Scripts\activate
```
### MacOS/Linux
```bash
source venv/bin/activate
```
